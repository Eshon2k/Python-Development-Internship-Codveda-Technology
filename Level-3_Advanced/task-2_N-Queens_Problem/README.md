## Level 3 - Task 3: N-Queens Problem
A solver for the classic N-Queens puzzle using a recursive backtracking algorithm.
- **Board Representation**: Utilizes a 2D array to represent the $N \times N$ chessboard.
- **Backtracking**: Implements an efficient search to place queens in safe positions.
- **Constraint Logic**: Ensures no two queens share the same row, column, or diagonal.
